# DraftOne Studio Engineering — Free Starter Site

This pack contains a simple one‑page website ready for **GitHub Pages (free)**.

## How to publish (free)
1) Create a GitHub account (free) at https://github.com
2) Click **New repository** → name it: `draftone-site`
3) Upload these files (drag & drop the `assets/`, `index.html`, `styles.css`)
4) Go to **Settings → Pages** → **Build and deployment** = *Deploy from a branch*
5) Select branch `main` and folder `/ (root)` → Save
6) Your site will be live at: `https://<your-username>.github.io/draftone-site/`

### Optional
- Replace the contact button URL: edit `index.html` where it says `https://tally.so/r/your-form` with your actual Tally form link.
- Replace `mailto:` with your chosen email (Gmail/Proton).

## Edit text and colors
- Colors are in `styles.css` (`--primary`, `--accent`, `--neutral`)
- Logo and icon are in `/assets`

## Notes
- This is a static site (fast, secure, free hosting).
- You can later connect a **custom domain** when you’re ready to pay for one.
